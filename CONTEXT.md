# SupportKos Bali - Konteks Bisnis

## Konsep Bisnis
Jasa "Inspeksi Kos Pihak Ketiga" di Bali.
Bukan agen pencari kos, BUKAN calo. Hanya jasa verifikasi independen.

## Target Pasar
Orang dari luar Bali (Jakarta, Bandung, Surabaya) yang ingin pindah sementara ke Bali,
namun tidak berani DP kos langsung karena foto/info di internet tidak bisa dipercaya.

## Layanan / Produk

### 1. Kos Inspection Report - Rp 150.000/kos
- Klien cari kos sendiri di internet/grup FB
- Kirim link ke tim kami
- Kami datang ke lokasi dan buat laporan:
  - Speed test WiFi (screenshot)
  - Cek sinyal operator (Telkomsel/XL/Smartfren)
  - Cek keran air (tekanan, bau, warna)
  - Foto/video kondisi real kamar (tanpa filter/edit)
  - Foto lingkungan sekitar (keamanan, akses motor, dll)
- Hasil dikirim dalam 24 jam (PDF Report + Video)

### 2. Soft-Landing Starter Kit (Upsell) - Rp 150.000 - 300.000 (jasa, belum termasuk harga barang)
- Pembelian & pengantaran galon air + pompa
- Token listrik awal
- Perlengkapan kebersihan dasar (sapu, pel, sabun)
- Opsional: Jemput dari Bandara Ngurah Rai ke kos

## Strategi Marketing
- Iklan TikTok/IG Ads target kota: Jakarta, Bandung, Surabaya
- Kata kunci: "Pindah ke Bali" / "Cari kos Bali" / "scam kos Bali"
- Konten Faceless: Rekam layar HP saat video inspeksi kos sambil narasi masalah umum

## Tech Stack Website (Rencana)
- HTML / CSS / Vanilla JS (Single Page, cepat, tanpa framework berat)
- Form pemesanan terhubung ke WhatsApp
- Payment via QRIS / Midtrans (fase 2)
- Desain: Dark Mode, Glassmorphism, premium aesthetic
